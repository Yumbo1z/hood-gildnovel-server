Overview

Welcome to Hood Gild Novel — a roleplay server focused on governments, kingdoms, and gangs. This server is designed to simulate street life as well as government and military power. Form factions, run politics, hold events, and engage in conflict. PvP is allowed throughout the world, but every region follows its own in-game laws — roleplay should respect the country's laws where you act. Griefing and destruction are prohibited unless explicitly allowed by local law or in specially designated zones.

Features

  • Roleplay governments
  • Kingdoms & Gang factions
  • Economy & trade
  • Political events & roleplay scenarios
  • PvP allowed (subject to local in-game laws)
JAVA ONLY
Client-side Forge modpack
This server requires a custom client modpack on Java Edition. Download the pack and install Forge for the matching Minecraft version.
Steps: install the Forge installer for the server Minecraft version, run the installer to create a Forge profile, unzip the modpack and copy the contents into your .minecraft (or drop the mods folder into the Forge profile), then run Minecraft with the Forge profile.

Server Rules

  1. No griefing or arbitrary destruction of builds — respect other players' creations. PvP is allowed, but actions are governed by the laws of the country/region you are in; griefing is only permitted where local law explicitly allows it (for example in designated lawless zones).
  2. No cheating, exploits, or banned mods.
  3. Keep roleplay in-character where appropriate and be respectful in chat and on Discord.
  4. Follow staff instructions and server laws; abuse of roleplay (e.g., meta-gaming) may be moderated.
